## Running the Application

To run the application, follow the steps below:

### Running the Client Side

1. Navigate to the `client` directory:
   ```bash
   cd client
   ```
2. Run this command:
   ```bash
   npm i
   ```
3. Start the client:
   ```bash
   npm run start
   ```

### Running the Server Side

1. Navigate to the `server` directory:
   ```bash
   cd server
   ```
2. Run this command:
   ```bash
   npm i
   ```
3. Start the server:
   ```bash
   npm run start
   ```
